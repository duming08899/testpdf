# PDF 图片提取工具

从 PDF 文件中提取并压缩图片的桌面工具，支持单个文件和批量处理。

## 功能

- 提取 PDF 中嵌入的图片
- 图片压缩（JPEG/PNG/WEBP）
- 宽度限制缩放
- 保留原图选项
- 批量处理整个文件夹

## 下载

> 通过 [GitHub Actions](https://github.com/duming08899/testpdf/actions) 自动打包。

- **macOS**: 下载 Artifact → `PDF图片提取工具_macOS`
- **Windows**: 下载 Artifact → `PDF图片提取工具_Windows`

## 本地运行

```bash
pip install pymupdf pillow
python pdf_image_extractor_gui.py
```
