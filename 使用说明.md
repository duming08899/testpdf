# PDF 图片提取压缩工具 - 使用说明

## 功能特点
- 📤 从 PDF 文件中提取所有内嵌图片
- 📦 支持图片压缩（JPEG / PNG / WebP 格式）
- 🔧 可调节压缩质量、限制最大尺寸
- 📁 支持单个 PDF 或批量处理整个文件夹
- 💻 纯 Python 实现，Windows 本地运行

## 环境要求
- Windows 系统
- Python 3.7 或更高版本

## 安装步骤

### 1. 安装 Python
如果还没安装 Python，去官网下载：
https://www.python.org/downloads/windows/

> 安装时记得勾选 "Add Python to PATH"

### 2. 安装依赖库
打开命令提示符（CMD）或 PowerShell，进入脚本所在目录，执行：

```bash
pip install -r requirements.txt
```

或者手动安装：

```bash
pip install pymupdf pillow
```

## 使用方法

### 基础命令格式
```bash
python pdf_image_extractor.py [PDF文件路径] [选项]
```

### 常用示例

#### 1. 仅提取图片（不压缩）
```bash
python pdf_image_extractor.py "C:\Users\Desktop\test.pdf" --extract-only
```
> 提取的图片会保存在 `test_extracted_images` 文件夹中

#### 2. 提取并压缩（推荐）
```bash
python pdf_image_extractor.py "C:\Users\Desktop\test.pdf" -q 85
```
> `-q 85` 表示压缩质量为 85（1-100，越大质量越好）

#### 3. 提取并限制图片尺寸
```bash
python pdf_image_extractor.py "C:\Users\Desktop\test.pdf" -q 80 --max-width 1920
```
> 宽度超过 1920px 的图片会等比例缩小

#### 4. 转换为 WebP 格式（体积最小）
```bash
python pdf_image_extractor.py "C:\Users\Desktop\test.pdf" -q 80 --format webp
```

#### 5. 批量处理整个文件夹的 PDF
```bash
python pdf_image_extractor.py --folder "C:\Users\Desktop\pdfs" -q 80
```

#### 6. 保留原始图片，压缩版另存
```bash
python pdf_image_extractor.py "C:\Users\Desktop\test.pdf" -q 85 --keep-original
```

#### 7. 直接压缩已有图片文件夹
```bash
python pdf_image_extractor.py --compress-folder "C:\Users\Desktop\images" -q 80
```

## 参数说明

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `pdf` | PDF 文件路径（位置参数） | - |
| `--folder, -f` | 批量处理的 PDF 文件夹 | - |
| `--output, -o` | 输出目录 | 与 PDF 同目录 |
| `--extract-only` | 仅提取不压缩 | 关闭 |
| `--quality, -q` | 压缩质量 1-100 | 85 |
| `--max-width` | 最大宽度（像素） | 不限制 |
| `--max-height` | 最大高度（像素） | 不限制 |
| `--format` | 输出格式: jpg/png/webp | 保持原格式 |
| `--keep-original` | 保留原图，压缩版另存 | 关闭（覆盖） |
| `--compress-folder` | 直接压缩图片文件夹 | - |

## 输出文件说明

### 提取模式
```
你的PDF文件.pdf
└── 你的PDF文件_extracted_images/
    ├── 你的PDF文件_page001_img01.png
    ├── 你的PDF文件_page001_img02.jpg
    ├── 你的PDF文件_page002_img01.png
    └── ...
```

### 提取 + 压缩（不保留原图）
```
你的PDF文件.pdf
└── 你的PDF文件_extracted_images/  ← 图片被直接压缩覆盖
    └── ...
```

### 提取 + 压缩（保留原图）
```
你的PDF文件.pdf
├── 你的PDF文件_extracted_images/    ← 原始图片
└── 你的PDF文件_compressed_images/   ← 压缩后图片
```

## 推荐设置

| 场景 | 推荐参数 | 说明 |
|------|----------|------|
| 文档类 PDF | `-q 80` | 兼顾清晰度和体积 |
| 扫描件 PDF | `-q 85 --max-width 1920` | 文字清晰，体积适中 |
| 图片类 PDF | `-q 90` | 高质量，体积较大 |
| 网页展示用 | `-q 80 --format webp` | 体积最小，加载快 |
| 批量归档 | `-q 70 --max-width 1280` | 最小体积，可预览 |

## 常见问题

### Q: 运行提示 "找不到模块"
A: 重新安装依赖：`pip install pymupdf pillow`

### Q: 提取的图片数量比预期少
A: 有些 PDF 的"图"实际是矢量图形，不是内嵌图片，这种无法提取。可以用截图的方式获取。

### Q: 压缩后图片变大了？
A: 如果原图已经是高度压缩的，再次压缩可能反而变大。这种情况建议使用 `--extract-only` 仅提取。

### Q: WebP 格式怎么打开？
A: Windows 10/11 自带照片应用可以直接打开，或者用浏览器打开。

### Q: 如何批量处理多个文件夹？
A: 写一个 bat 批处理文件，例如：
```batch
@echo off
for /d %%d in (*) do (
    python pdf_image_extractor.py --folder "%%d" -q 80
)
```


## 打包成 EXE（无需 Python 环境也能运行）

### 方法一：一键打包（推荐）
直接双击运行 `一键打包.bat`，会自动安装依赖并打包成 `PDF图片提取工具.exe`

### 方法二：手动打包
在命令行中执行：
```bash
pip install pyinstaller
pyinstaller --onefile --console --name "PDF图片提取工具" pdf_image_extractor.py
```

打包完成后，exe 文件在 `dist` 文件夹中。

### EXE 使用方法
```bash
# 提取图片
PDF图片提取工具.exe test.pdf --extract-only

# 提取并压缩
PDF图片提取工具.exe test.pdf -q 85

# 批量处理
PDF图片提取工具.exe --folder D:\pdfs -q 80
```

> 注意：打包后的 exe 只在同版本 Windows 上运行，Win10 打包的可以在 Win10/Win11 上运行。

## GUI 图形界面版

### 启动 GUI 版
```bash
python pdf_image_extractor_gui.py
```

### GUI 功能说明
- **选择输入**：单个 PDF 文件 或 整个文件夹批量处理
- **压缩质量**：滑块调节 10-100，越大越清晰
- **限制宽度**：超过指定宽度自动等比缩小
- **保留原图**：压缩图片另存，不覆盖原图
- **一键打开输出文件夹**：处理完直接打开结果目录

### 打包 GUI 版成 EXE
双击运行 `一键打包_GUI版.bat`，自动生成无命令行窗口的 GUI 程序。

> --windowed 参数打包后不会显示黑色命令行窗口，更美观。
